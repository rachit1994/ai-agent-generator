# Content, Facilitation, and Community (human lifecycle)

Compiles to persona YAML stages -> steps with low-variance discussion quality and repeatable community growth.

## Always-on rules
- No session starts without approved question ladder and moderation plan.
- Prompts must pass policy/sponsor boundary check before use.
- All publish and outreach actions must be logged in source systems same day.
- If gate fails, stop the phase, log blocker, escalate to owner.

## Phase process (intern-safe, anti-bug)
- Phase -1 - Content positioning. Actions: browse trusted sources (approved industry sites + prior feedback docs), update content matrix, verify with editor. Gate: editorial plan approved. Artifacts: content matrix + source links.
- Phase 9 - Moderator preparation. Actions: assemble brief from template, run rehearsal script, capture deviations, verify with producer. Gate: rehearsal complete. Artifacts: moderator brief + rehearsal log.
- Phase 12 - Facilitation delivery. Actions: follow timed discussion map, log insights live, tag off-topic risks, escalate policy issues immediately. Gate: facilitator quality score met. Artifacts: facilitation log.
- Phase 16 - Knowledge portal. Actions: publish replay/summary/resources via checklist, run link/asset QA, verify publish SLA clock. Gate: publish SLA met. Artifacts: portal publish checklist.
- Phase 18 - Community activation. Actions: send follow-ups from approved templates, record replies/referrals in CRM, verify outreach totals. Gate: engagement target met. Artifacts: community activity log.
- Phase 19 - Topic feedback. Actions: collect survey + call notes, cluster recurring themes, update next-topic backlog, verify weekly refresh. Gate: backlog updated weekly. Artifacts: topic revision notes.

## Daily execution checklist
1. Browse only approved sources; paste links in today log.
2. Perform system updates (CRM/portal/docs) immediately after each action.
3. Run automation only from existing scripts/templates; no ad-hoc code edits.
4. Complete gate check, then request verifier sign-off.
5. Escalate unresolved blocker within 15 minutes.
