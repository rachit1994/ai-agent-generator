## Purpose

Enhance pre- and in-event personalization for delight.

## Teams

Concierge.

## Tools

- Affinity (connection mapping).
- Slack (private channels).
- SavvyCal (ad-hoc scheduling).

## Process

```
Pre-event check-in (preferences survey)
↓
Real-time support (chat moderation)
↓
Connection facilitation (breakout intros)
↓
Post-event debrief (feedback capture)
```

## Deliverables

- Curated connections (3–5 peer matches).
- Private executive groups (Slack/Circle invites).

## Metrics

| Metric                   | Target |
|--------------------------|--------|
| Executive satisfaction   | >9/10  |

## Success Criteria

>80% report new relationships formed.
