# Phase 0: Strategic Market Signal Engine

[← Phase -1](phase--1.md) | [Overview](00-overview.md) | [Phase 1 →](phase-01.md)

---

## Purpose

Detect emerging executive pain points via signals and convert them into high-relevance event topics.

## Teams

Strategy, Data Engineering, AI Intelligence.

## Tools

- BigQuery (signal aggregation).
- Python NLP (sentiment analysis).
- OpenAI API (problem summarization).
- Clay (data orchestration).
- Bombora (intent signals).
- CB Insights (trend reports).
- Gartner data (benchmarking).

## Process

```
Market signals aggregation (news, forums, earnings calls)
↓
Problem extraction (NLP for themes like "AI governance")
↓
Topic clustering (vector similarity for grouping)
↓
Sponsor alignment (match to partner verticals)
↓
Event topic creation (refined briefs with personas)
```

## Deliverables

- Weekly topic backlog (20–30 ideas, prioritized).
- Sponsor opportunity map (heatmaps by industry).
- Executive persona targeting (5–7 archetypes per topic).

## Metrics

| Metric                  | Target    |
|-------------------------|-----------|
| Validated topics/week   | 10        |
| Sponsor alignment score | >80%      |
| Executive relevance rating | >8/10   |

## Success Criteria

Topics result in >75% acceptance rates; signals predict 20% of future sponsor inquiries.
