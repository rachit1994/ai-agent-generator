# Phase 14: Event QA & DEI Metrics

[← Phase 13](phase-13.md) | [Overview](00-overview.md) | [Phase 15 →](phase-15.md)

---

## Purpose

Measure and iterate on quality, inclusivity, and performance.

## Teams

Event Operations, DEI Specialist.

## Tools

- Typeform (NPS surveys).
- Google Forms (DEI self-reports).
- Airtable (scorecard templates).

## Process

```
Feedback collection (post-event)
↓
Metric aggregation (dashboards)
↓
Root-cause analysis (low scores)
↓
Action logging (improvements)
```

## Metrics

| Metric                | Target   |
|-----------------------|----------|
| Executive NPS         | >8.5     |
| Participation rate    | >90%     |
| DEI balance           | >40% underrepresented groups |
| Moderator performance | >8/10    |

## Deliverables

- QA scorecard (event-grade).
- DEI cohort report (demographic breakdowns).

## Success Criteria

Quarterly NPS trend +0.5; zero bias incidents.
