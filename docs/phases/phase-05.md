# Phase 5: Gatekeeper & EA Relationship Management

[← Phase 4](phase-04.md) | [Overview](00-overview.md) | [Phase 6 →](phase-06.md)

---

## Purpose

Navigate executive assistants (EAs) and gatekeepers for seamless scheduling.

## Teams

Concierge Liaison.

## Tools

- Salesforce EA database (role tagging).
- Sendoso (gifting for rapport).
- Gmail automation (personalized sequences).

## Process

```
EA identification (LinkedIn, company directories)
↓
Rapport building (value-add intros, resources)
↓
Double-touch outreach (email + call)
↓
Calendar confirmation (shared links, buffers)
```

## Deliverables

- EA database (500+ profiles, rapport scores).
- Confirmed attendance logs (with backups).

## Metrics

| Metric       | Target |
|--------------|--------|
| No-show rate| <10%   |

## Success Criteria

>95% confirmation within 48 hours; EAs as referral sources.
