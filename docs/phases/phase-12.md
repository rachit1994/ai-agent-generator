# Phase 12: Event Execution

[← Phase 11](phase-11.md) | [Overview](00-overview.md) | [Phase 13 →](phase-13.md)

---

## Purpose

Run high-fidelity roundtables with structured flow.

## Teams

Event Operations, Technical Production, Concierge.

## Tools

- Zoom Events (hosting).
- Mentimeter (polling).
- Otter.ai (live notes).

## Standard Agenda (60 minutes)

```
0–5: Introductions (icebreaker round)
↓
5–15: Framing insight (moderator keynote)
↓
15–40: Moderated discussion (Q&A probes)
↓
40–50: Peer exchange (breakouts)
↓
50–60: Key insights synthesis (group commitments)
```

## Process

```
Kickoff (attendance ping)
↓
Agenda execution (timekeeping)
↓
Interaction capture (chat logs, polls)
↓
Wrap-up (NPS pulse)
```

## Deliverables

- Insights summary (1-page recap).
- Relationship signals (co-participation data).

## Metrics

| Metric             | Target |
|--------------------|--------|
| Participation rate | >90%   |
| Session completion | 100%   |

## Success Criteria

Average engagement time >50 minutes; NPS >8.5 immediate.
