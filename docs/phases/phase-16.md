## Purpose

Distribute event value on-demand for ongoing engagement.

## Teams

Content Studio, Community.

## Tools

- Teachable / Kajabi (LMS platform).
- Cloudinary (media hosting).
- Auth0 (secure access).

## Process

```
Content upload (replays, transcripts)
↓
Tagging and search optimization
↓
Personalization (recommendations)
↓
Usage tracking (analytics)
```

## Deliverables

- Replay library (searchable archives).
- Research insights packs (curated downloads).
- Personalized playlists (AI-generated).

## Metrics

| Metric             | Target    |
|--------------------|-----------|
| Replay engagement  | >30%      |
| Portal usage       | >50% executives |

## Success Criteria

Portal drives 15% of repeat attendance.
