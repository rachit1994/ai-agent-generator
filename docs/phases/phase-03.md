# Phase 3: Sponsor Portfolio Strategy

[← Phase 2](phase-02.md) | [Overview](00-overview.md) | [Phase 4 →](phase-04.md)

---

## Purpose

Build a diversified sponsor base with tiered commitments for predictable revenue.

## Teams

Sales, Sponsor Success, Finance.

## Tools

- Salesforce (opportunity tracking).
- ZoomInfo (prospect research).
- Apollo (email sequencing).
- Outreach (sales automation).
- DocSend (pitch decks).
- Zuora billing (subscription management).

## Sponsor Tiers

| Tier             | Description                          | Commitment          |
|------------------|--------------------------------------|---------------------|
| Strategic Partner| Annual contract, multi-event access  | $100K–$500K/year    |
| Topic Sponsor    | Event-specific branding and intros   | $5K–$15K/event      |
| Pilot Sponsor    | Experimental formats, beta testing   | $2K–$5K/trial       |

## Process

```
Prospect identification (ICP: VCs, consultancies)
↓
Value prop tailoring (ROI case studies)
↓
Negotiation and close (custom SLAs)
↓
Onboarding (contract execution, kickoff call)
↓
Portfolio balancing (diversify by industry)
```

## Deliverables

- Sponsor contracts (templated, e-signed).
- Event commitments calendar (booked slots).
- Revenue forecast model (quarterly projections).

## Metrics

| Metric          | Target     |
|-----------------|------------|
| Sponsor renewal | >80%       |
| Event revenue   | $25K–$40K  |
| Gross margin    | 60–70%     |

## Success Criteria

Portfolio yields 3x YoY growth; >90% of revenue recurring.
