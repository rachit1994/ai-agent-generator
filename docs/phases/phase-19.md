# Phase 19: Topic Feedback Engine

[← Phase 18](phase-18.md) | [Overview](00-overview.md) | [Phase 20 →](phase-20.md)

---

## Purpose

Harvest post-event signals to refine future topics.

## Teams

Data & AI, Strategy.

## Tools

- Python NLP (feedback parsing).
- BigQuery (storage).
- OpenAI (sentiment clustering).

## Process

```
Feedback ingestion (NPS, comments)
↓
Theme extraction (e.g., "supply chain gaps")
↓
Backlog integration (priority scoring)
↓
Validation loop (A/B testing)
```

## Deliverables

- New topic backlog (refined 10–15).
- Feedback synthesis report.

## Metrics

| Metric              | Target |
|---------------------|--------|
| Feedback response rate | >70% |
| Cycle time to new topic | <1 week |

## Success Criteria

Feedback improves relevance by 10% QoQ.
