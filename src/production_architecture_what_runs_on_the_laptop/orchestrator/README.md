# Orchestrator (`src/production_architecture_what_runs_on_the_laptop/orchestrator`)

Aligned to **`docs/AI-Professional-Evolution-Master-Architecture.md`** (§21 target layout): the **SDE orchestrator** Python package lives here (import name **`orchestrator`**). Runnable pipeline and modes live in **`workflow_pipelines`** (`production_pipeline_task_to_promote`, `execution_modes`); gates in **`guardrails_and_safety`**; shared primitives in repo-root **`libs/`** (see [`libs/README.md`](../../../libs/README.md)).

| Path | Role |

| `api/` | **Public import surface** — `from orchestrator.api import execute_single_task`, `run_benchmark`, `generate_report`. |
