# Folder: `src/evaluation_framework/online_evaluation_shadow_canary_artifact`

## Why this folder exists
This folder provides deterministic local online-evaluation shadow/canary artifact generation and validation.

## What is present
- `__init__.py` (file): Public exports for runtime and contracts.
- `contracts.py` (file): Contract constants and fail-closed validators.
- `runtime.py` (file): Deterministic shadow/canary derivation.
- `surface.py` (file): Feature-surface metadata.
- `tests/` (folder): Focused runtime and contract tests.
